Every serious Claude Code setup lives or dies by its `CLAUDE.md` — the file the agent re-reads on every turn. Left alone it bloats into a thousand-line manual that costs tokens, buries the rules that matter, and stops your teammates from reading it. This series treats it as what it actually is: a contract. Start with the 30-minute cleanup, then go deep on the mechanisms that keep it that way.
